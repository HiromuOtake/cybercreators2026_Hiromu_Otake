◆作品について◆

◇使用言語 : C++
◇SDK      : DirectX9

◆動作確認済み環境◆
Windows11
（DirectxInput対応のゲームパッド(ジョイスティック)に対応しています。）

◆操作方法◆
キーボード
 AD			: 左右移動
 SPACE		: ジャンプ
 K			: クローン選択状態に遷移
 十字キー	: クローンを選択
 ENTER		: クローンを生成

ゲームパッド
 左スティック	: 左右移動
 十字キー		: クローンを選択
 Aボタン		: ジャンプ/クローンを生成
 Xボタン		: クローン選択状態に遷移
